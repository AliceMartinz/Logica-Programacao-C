## 📂 Projetos em Destaque

### 🌿 Sistema de Gestão de Plantio (Agro)
Um sistema de linha de comando para gerenciar a produção agrícola.
* **Recursos:** Cadastro de plantas, comparativo de códigos, cálculos de média de gastos, identificação de maior produção por período e atualização de dados.
* **Conceitos aplicados:** `structs`, loops (`do-while`, `for`), condicionais complexas e acumuladores.

### 🚗 Gerenciador de Garagem e Veículos
Software simples para controle de inventário de veículos.
* **Recursos:** Cadastro de modelos, anos e consumo; busca pelo veículo de maior valor e cálculo de média de mercado.
* **Conceitos aplicados:** Vetores de estruturas, busca em arrays e formatação de saída de dados.
